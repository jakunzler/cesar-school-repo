# CESAR School Repository

Bem-vindos(as)! Este repositório centraliza materiais de apoio das disciplinas ministradas na CESAR School.

## TL;DR (leitura rápida)

- Este repositório reúne **documentos, bibliografia e guias práticos** da disciplina.
- Os **códigos iniciais** já foram submetidos.
- O **código completo será adicionado gradualmente** ao longo da disciplina.
- Use este espaço como referência oficial para acompanhar aulas, estudos e atividades.

## O que você vai encontrar aqui

- `oran/docs/`: documentos da disciplina (plano, roadmap e materiais de apoio).
- `oran/bibliography/`: bibliografia básica e complementar.
- `oran/code/`: estrutura dos projetos e ambientes práticos (conteúdo em evolução).
- `slides/`: slides das aulas (conteúdo em evolução).

## Status do conteúdo

No momento, os arquivos iniciais de código já estão disponíveis para organização e contexto.
As implementações e exemplos práticos serão publicados posteriormente, conforme o andamento das aulas.

## Como aproveitar melhor este repositório

1. Comece pelos documentos em `oran/docs/`.
2. Consulte a bibliografia em `oran/bibliography/` para aprofundamento.
3. Acompanhe as atualizações em `oran/code/` durante a disciplina.
4. Acompanhe as atualizações em `slides/` durante a disciplina.

## Como contribuir

1. Fork este repositório.
2. Crie uma branch para sua contribuição.
3. Faça suas alterações.
4. Faça um pull request para o repositório principal.

## Atualizações

Este repositório será atualizado continuamente. Recomendamos sincronizar com frequência para não perder novos materiais.
